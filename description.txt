Hãy xây dựng một hệ thống backend của một phần mềm quản lý các thiết bị máy tính của trẻ em, mục tiêu là đơn giản,
hỗ trọ tốt self host cho nên tính năng đăng nhập đơn giản chỉ username và password, username và password mặc định
ban đầu được thiết lập bởi biến môi trường, chỉ có tính năng đổi mật khẩu. Các tính năng của phần mềm này
1. Đăng nhập / đổi mật khẩu đơn giản
2. Hiển thị các thiết bị con đang kết nối tới server
3. Cho phép phụ huynh gửi lệnh tắt máy, khởi động lại máy, khóa máy, mở khóa máy (web socket)
    - Khi gửi các lệnh như khóa, mở khóa, tắt máy, khởi động sẽ kèm theo khoảng thời gian x giây
    - Ví dụ: lock:3600 khóa 1 tiếng, unlock:3600 mở khóa sau 1 tiếng, shutdown:3600 tắt máy sau 1 tiếng, restart:3600 khởi động lại sau 1 tiếng
    - Mặc định sẽ là sau 1 phút (60 giây)
    - Set giá trị bé hơn hoặc bằng 0 thì sẽ thực hiện ngay lập tức
    - Các tính năng schedule sẽ thực hiện phía client, backend ko xử lý schedule gì cả
4. Lưu lại thời gian mở máy, tắt máy, khóa máy, mở khóa máy, khởi động lại máy
5. Thông tin thiết bị bao gồm tên thiết bị, địa chỉ ip
6. Quản lý trạng thái thiết bị: offline, online và trạng thái lock, unlock. Shutdown restart không cần lưu lại trạng thái
7. Admin có thể gửi tin nhắn tói client và client sẽ hiển thị qua window notification của windows nếu đang unlock

Cách client kết nối tới server:
Client sẽ kết nối tới server thông qua websocket, sau khi kết nối thành công thì client sẽ gửi thông tin thiết bị lên server
Client sẽ gửi các thông tin sau:
- Tên thiết bị
- Địa chỉ ip
- DeviceID
DeviceID sẽ không thay đổi, server dựa vào DeviceID để quản lý thiết bị

Admin làm việc qua giao diện web, rest api.

Tech stack:
- Java 25 (tận dụng virtal thread nếu cần thiết)
- Spring boot 4.0
- Postgres
- Websocket
- Rest api
- Swagger
- Auth dùng cơ chế đơn giản nhất không cần JWT, có thể dùng session cho nhanh

Tạo file README.md để hướng dẫn cách build, run, deploy, config biến môi trường, cách client kết nối tới server (client có thể  viết bằng C++ hoặc bất kì)
Tạo scripting hướng dẫn xây dựng frontend web dành cho admin sử dụng Angular và Tailwind CSS
Tạo scripting hướng dẫn xây dựng client sử dụng C++ (windows 11) và có thể chạy ngầm, auto start khi khởi động máy và sau khi mạng được kết nối thành công
